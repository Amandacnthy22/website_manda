# AI-Ready Proyek Website 5 Halaman (Fotografi Sederhana)

Repositori ini berisi template proyek HTML + CSS untuk uji kinerja (Asesmen Sumatif Akhir Semester Genap).

## Ketentuan Wajib (ringkas)
- 5 halaman HTML: `index.html`, `profil.html`, `galeri.html`, `artikel.html`, `kontak.html`
- 1 file CSS eksternal: `style.css`
- Navigasi konsisten antar halaman via `<nav>` (Flexbox).
- Minimal 1 gambar di setiap halaman, dan `galeri.html` minimal 4 gambar dengan caption.
- Struktur semantik: `<header>`, `<main>`, `<footer>`
- CSS minimal 5 selector berbeda: tag, class, id, dan pseudo-class (`:hover`) minimal.

## Asset
Gambar tersedia di folder `assets/`:
- `assets/photograph.jpg`
- `assets/mountain.jpg`
- `assets/bali.jpg`
- `assets/food.jpg`
- `assets/art.jpg`
- `assets/product.jpg`

## Cara Pakai
Buka salah satu halaman (mis. `index.html`) di browser. Pastikan folder `assets/` dan file `style.css` berada pada level yang sama.
